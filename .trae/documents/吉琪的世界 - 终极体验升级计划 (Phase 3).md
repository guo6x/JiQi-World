# 吉琪的世界 (JiQi's World) - 终极体验升级计划 (Phase 3)

根据您的详细反馈，我制定了全面的升级计划。我们将彻底重塑交互方式，引入**虚拟光标系统**，并还原真实的 DNA 双螺旋结构。

## 1. 视觉重塑：真实的 DNA 双螺旋 (True DNA Helix)
*   **结构还原**：
    *   **双链骨架**：现在的照片将严格分布在两条平行的螺旋链上。
    *   **碱基对连线 (Base Pairs)**：在两条链对应的照片之间绘制发光的连接线，模拟 DNA 的碱基配对结构。
    *   **光效增强**：连线将带有微弱的流动光效，增加科幻感。
*   **照片优化**：
    *   **尺寸增大**：全局放大照片尺寸，让细节更清晰。
    *   **日期标签**：在每张照片下方添加极简的日期显示（如 "2023.12"），增强纪念意义。

## 2. 交互革命：手势虚拟光标系统 (Virtual Cursor System)
为了解决"剪刀手点不到"的问题，我们将引入类似 Xbox Kinect 的**虚拟光标**：
*   **👆 食指光标 (Pointing)**：
    *   只要手指出现在画面中，屏幕上就会出现一个跟随食指尖移动的光亮光标。
    *   您可以像使用鼠标一样，用手指精准地"指"向任何一张照片。
*   **🤏 捏合点击 (Pinch to Click)**：
    *   当光标悬停在照片上时，**捏合拇指和食指**（模拟点击动作），即触发选中/打开。
    *   再次捏合即可关闭/退出详情。
*   **🖐 旋转与缩放 (Rotate & Zoom)**：
    *   **单手张开 (Open Palm)**：进入"控制模式"。此时移动手掌，世界会跟随您的手掌进行 **6轴自由旋转**。
    *   **双手捏合 (Two Hand Pinch)**：检测双手距离的变化来实现放大/缩小（如果单手难以操作，我们将简化为：握拳后上下移动手部来进行缩放）。

## 3. 手势逻辑梳理 (Gesture Refinement)
我们将重新定义一套互不冲突、符合直觉的手势集：

| 手势 | 动作 | 功能 |
| :--- | :--- | :--- |
| **👆 食指指点 (POINT)** | 移动食指 | **移动光标**。用于瞄准照片。 |
| **👌 捏合 (PINCH)** | 拇指食指捏合 | **点击/确认**。相当于鼠标左键点击。 |
| **🖐 张开手掌 (OPEN_PALM)** | 移动手掌 | **旋转视角**。像抓着地球仪一样转动世界。 |
| **✊ 握拳 (FIST)** | 上下移动 | **缩放视角**。上移放大，下移缩小。 |
| **👍 竖大拇指 (THUMB_UP)** | 保持 1秒 | **切换布局**。在球体和 DNA 之间切换。 |

## 4. 参数微调
*   **降速**：大幅降低自动旋转速度，让浏览更从容。
*   **防抖**：增强光标的平滑算法，消除手部微颤带来的抖动。
