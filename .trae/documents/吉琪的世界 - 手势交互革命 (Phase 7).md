# 吉琪的世界 (JiQi's World) - 手势交互革命 (Phase 7)

根据您对当前手势系统的严厉批评和具体需求，我制定了以下彻底的重构计划。我们将废弃那些“垃圾”手势，换上一套**符合直觉、高识别率且功能强大**的全新方案。

## 1. 全新定义的手势集 (The "Iron Man" Set)
我们将手势功能重新分配给那些最具辨识度的动作，确保“好用且好识别”。

| 功能 | 旧手势 (废弃) | **新手势 (New)** | **交互逻辑** |
| :--- | :--- | :--- | :--- |
| **光标移动** | 👆 POINT | **👆 食指 (POINT)** | (保留) 只要伸出食指，光标就跟随移动。这是最自然的光标操作。 |
| **点击/确认** | 👌 PINCH (捏合) | **✌️ 剪刀手 (VICTORY)** | **"胜利点击"**。当光标瞄准照片时，比出剪刀手即触发点击/打开。比捏合更稳定，识别率极高。 |
| **退出/返回** | 👌 PINCH (再次) | **👎 拇指向下 (THUMB_DOWN)** | **"否定/退出"**。拇指向下即表示关闭当前照片，回归原位。直观且不易误触。 |
| **六轴旋转** | 🖐 OPEN_PALM | **✊ 握拳 (FIST)** | **"抓取旋转"**。像抓住地球仪一样，握拳后移动拳头，世界会跟随拳头**上下左右全方位旋转**。 |
| **缩放视角** | ✊ FIST (上下) | **🖐 张开手掌 (OPEN_PALM)** | **"掌控缩放"**。张开手掌后，手掌**向上推**=放大(Zoom In)，**向下拉**=缩小(Zoom Out)。 |
| **切换布局** | 👍 THUMB_UP | **👍 竖大拇指 (THUMB_UP)** | (保留) 经典的切换手势，维持“一次一触发”的稳定逻辑。 |
| **管理员** | 3-2-1 | **👌 OK 手势 (OK)** | **"OK 确认"**。为了简化，我们将复杂的 3-2-1 序列替换为一个特殊的 **OK 手势** (拇指食指相扣，其余三指伸直) 维持 2 秒触发。 |

## 2. 功能实现细节
*   **六轴自由旋转 (True 6-Axis)**:
    *   **实现原理**: 当检测到 **✊ 握拳** 时，我们将拳头在屏幕上的位置 `(x, y)` 直接映射到摄像机的**方位角 (Azimuth)** 和 **极角 (Polar)**。
    *   **效果**: 您的拳头就像一个虚拟摇杆，指哪转哪，彻底告别单平面旋转的尴尬。
*   **自动旋转**:
    *   **提速**: 基础自转速度将提升 30%，达到您要求的“快一点点”。
*   **防抖**:
    *   所有连续手势（光标、旋转、缩放）都将加入**加权平滑算法 (Lerp)**，消除手部微颤。

## 3. 稳定性保证
*   **手势互斥**: 严格的状态机逻辑，确保不会同时触发旋转和缩放。
*   **平滑过渡**: 退出手动控制时，自动旋转会平滑介入，不会有顿挫感。
